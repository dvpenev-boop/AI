# BuildPro Construction Website

A modern, responsive construction services website built with HTML5 and CSS3.

## Features

- **Single-page layout** with smooth scrolling navigation
- **Fully responsive design** that looks great on desktop, tablet, and mobile devices
- **Modern CSS Grid and Flexbox** layouts
- **Professional construction company branding**
- **Clean, accessible design** following web standards
- **No JavaScript dependencies** - pure HTML/CSS implementation

## Sections

1. **Home (Hero)** - Eye-catching header with company mission and call-to-action buttons
2. **Services** - Four main service categories with detailed features
3. **Promotions** - Special offers and deals for customers
4. **News** - Latest company updates and industry insights
5. **Contacts** - Complete contact information and quote request form

## Design Features

- **Color Scheme**: Orange gradient (#ff6b35 to #f7931e) with neutral grays
- **Typography**: Roboto for body text, Oswald for headings
- **Icons**: Font Awesome 6.4.0 for professional iconography
- **Images**: High-quality stock photos from Unsplash
- **Animations**: Subtle CSS animations and hover effects
- **Accessibility**: Semantic HTML and proper contrast ratios

## Browser Support

- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+

## Mobile Responsiveness

The website is fully responsive with breakpoints at:
- Desktop: 1200px+
- Tablet: 768px - 1199px
- Mobile: 480px - 767px
- Small mobile: <480px

## Setup Instructions

1. Download all files to your web server directory
2. Open `index.html` in a web browser
3. No additional setup required - all dependencies are loaded via CDN

## Customization

To customize the website:

1. **Colors**: Modify the CSS custom properties in `styles.css`
2. **Content**: Edit the HTML content in `index.html`
3. **Images**: Replace Unsplash URLs with your own images
4. **Contact Info**: Update contact details in the footer and contact sections

## Performance

- **Optimized images**: All images are properly sized and compressed
- **Minimal external requests**: Only essential CDN resources
- **Clean CSS**: Efficient selectors and minimal redundancy
- **Fast loading**: Lightweight codebase with no JavaScript
