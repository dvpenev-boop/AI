# Баба Пена - Счетоводна Фирма

Добре дошли в сайта на "Баба Пена" - вашият надежден партньор в счетоводството!

## Описание на проекта

Този проект представлява уебсайт за счетоводна фирма "Баба Пена". Сайтът е проектиран да бъде отзивчив и да работи на различни устройства, включително телефони, лаптопи и компютри. 

## Структура на сайта

Сайтът включва следните основни секции:

- **Начало**: Въведение в услугите и философията на фирмата.
- **Услуги**: Подробна информация за счетоводните услуги, които предлагаме.
- **Цени**: Информация за ценовата структура на нашите услуги.
- **Оферти**: Актуални промоции и специални предложения за клиенти.

## Инструкции за настройка

1. Клонирайте репозитория:
   ```
   git clone <URL на репозитория>
   ```

2. Отворете проекта в браузър, като отворите `src/index.html`.

3. Уверете се, че имате инсталирани необходимите зависимости, ако е необходимо.
